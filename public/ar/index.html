جاهز للعمل في 60 ثانية

ثلاث خطوات. بدون إعدادات. بدون تهيئة SMTP.

1

احصل على مفتاح API

سجّل واحصل على مفتاحك. بدون بطاقة ائتمان، بدون تحقق من البريد، بدون انتظار.

2

أنشئ صندوق بريد

استدعاء API واحد ينشئ صندوق بريد إلكتروني كامل الوظائف لوكيلك بعنوان فريد.

3

أرسل واستقبل

وكيلك يرسل رسائل البريد، ويستقبل الردود عبر Webhooks، ويتتبع سلاسل المحادثات الكاملة.

كل ما يحتاجه وكيلك

ليس ذكاءً اصطناعياً لبريدك. بل بريد إلكتروني لذكائك الاصطناعي.

📨

إرسال واستقبال

بريد إلكتروني ثنائي الاتجاه بالكامل. AWS SES للإرسال، خادم SMTP مدمج للاستقبال. معالجة غير متزامنة قائمة على الطوابير.

💬

تجميع تلقائي للمحادثات

تجميع محادثات متوافق مع RFC 5322. تُجمّع الرسائل تلقائياً عبر Message-ID و In-Reply-To و References.

⚡

إشعارات Webhooks فورية

إشعارات فورية عند وصول الرسائل. حمولات موقّعة بـ HMAC-SHA256 مع إعادة محاولة تلقائية وتراجع تدريجي.

🌐

نطاقات مخصصة

استخدم نطاقك الخاص مع تحقق تلقائي من DKIM و SPF. وكيلك يرسل من your-brand.com.

📎

المرفقات

ارفع وأرفق ملفات حتى 10 ميغابايت. تخزين مدعوم بـ S3 مع روابط تحميل موقّعة مسبقاً.

🛡️

تتبع الارتداد

معالجة تلقائية لأحداث SES للارتدادات والشكاوى وتأكيدات التسليم.

🔑

مصادقة بمفتاح API

مصادقة بسيطة بالرمز المميز. تخزين مُشفّر بـ SHA-256 مع تحديد معدل لكل حساب ولكل صندوق بريد.

🧩

MCP Skill

خادم MCP جاهز للاستخدام مع Claude و Cursor وأي وكيل ذكاء اصطناعي متوافق مع MCP. npx @agentsend/mcp.

📖

توثيق OpenAPI

واجهة Swagger UI تفاعلية مع توثيق كامل لـ API. جرّب النقاط النهائية مباشرة على /docs.

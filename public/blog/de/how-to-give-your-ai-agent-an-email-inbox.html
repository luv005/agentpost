So gibst du deinem KI-Agenten ein eigenes E-Mail-Postfach

AgentSend Team· 20. März 2026· 8 Min. Lesezeit
Lesen auf: EN ES FR DE JA ZH PT

KI-Agenten können Code schreiben, im Web surfen und komplexe Workflows ausführen — aber die meisten können noch immer keine einfache E-Mail empfangen. AgentSend löst dieses Problem. Es ist E-Mail-Infrastruktur, die speziell für Agenten entwickelt wurde: Stelle ein echtes Postfach programmatisch bereit, empfange Nachrichten via Webhook und antworte über die REST API.

Schritt 1: Konto erstellen und API-Key holen

Gehe zu agentsend.io/auth/signup und erstelle ein kostenloses Konto. Der kostenlose Plan ermöglicht bis zu 10 E-Mails pro Tag. Generiere anschließend einen API-Key unter Einstellungen → API-Keys.

Schritt 2: Ein Postfach erstellen

POST /inboxes
curl -X POST https://agentsend.io/inboxes \
  -H "Authorization: Bearer DEIN_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "mein-recherche-agent",
    "webhookUrl": "https://dein-server.com/webhooks/email"
  }'

Dein Agent hat jetzt eine echte E-Mail-Adresse: mein-recherche-agent@agentsend.io

Schritt 3: Eingehende E-Mails via Webhook verarbeiten

Wenn jemand eine E-Mail an deinen Agenten sendet, führt AgentSend einen HTTP POST an deine webhookUrl durch — mit dem vollständigen Nachrichteninhalt inklusive Absender, Betreff, Text und Anhängen.

Schritt 4: Das MCP-Skill nutzen (Claude-Agenten)

Falls dein Agent auf Claude läuft, installiere das AgentSend MCP-Skill. Claude erhält dann native Werkzeuge zum Senden von E-Mails, zur Postfachverwaltung und zum Lesen von Threads — ohne zusätzlichen Webhook-Code.

Alle verfügbaren Werkzeuge findest du in der MCP-Skill-Dokumentation.

Häufige Fragen

Kann ein KI-Agent eine eigene E-Mail-Adresse haben?

Ja. AgentSend ermöglicht es, ein echtes E-Mail-Postfach für jeden KI-Agenten programmatisch bereitzustellen — ohne menschliche Registrierung. Der Agent erhält eine echte @agentsend.io-Adresse (oder deine eigene Domain).

Mit welchen Agent-Frameworks ist AgentSend kompatibel?

AgentSend funktioniert mit jedem Framework, das HTTP-Anfragen stellen oder MCP-Tools nutzen kann — darunter Claude, LangChain, CrewAI, AutoGen und OpenAI Assistants.

Ist AgentSend kostenlos?

AgentSend hat einen kostenlosen Plan: bis zu 10 E-Mails pro Tag pro Postfach. Bezahlpläne starten bei 9 $/Monat für 1.000 E-Mails/Tag mit eigenen Domains.

Bereit, deinem Agenten ein Postfach zu geben?

Kostenloser Plan verfügbar. Keine Kreditkarte erforderlich. E-Mail-Adresse in unter 30 Sekunden aktiv.

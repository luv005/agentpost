Como Dar ao seu Agente de IA uma Caixa de Email Própria

Equipa AgentSend· 20 de março de 2026· 8 min de leitura
Ler em: EN ES FR DE JA ZH PT

Os agentes de IA podem escrever código, navegar na web e executar fluxos de trabalho complexos — mas a maioria ainda não consegue receber um simples email. AgentSend resolve este problema. É infraestrutura de email concebida especificamente para agentes: provisione uma caixa de email real programaticamente, receba mensagens via webhook e responda através da API REST.

Passo 1: Criar uma Conta e Obter uma Chave API

Aceda a agentsend.io/auth/signup e crie uma conta gratuita. O plano gratuito permite até 10 emails por dia. Após o registo, gere uma chave API em Definições → Chaves API.

Passo 2: Criar uma Caixa de Email (Inbox)

Uma inbox é o contentor do endereço de email do seu agente. Crie-a com uma única chamada API:

POST /inboxes
curl -X POST https://agentsend.io/inboxes \
  -H "Authorization: Bearer A_SUA_CHAVE_API" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "meu-agente-pesquisa",
    "webhookUrl": "https://o-seu-servidor.com/webhooks/email"
  }'

O seu agente tem agora um endereço de email real: meu-agente-pesquisa@agentsend.io

Passo 3: Processar Emails Recebidos via Webhook

Quando alguém envia um email para o seu agente, o AgentSend faz um HTTP POST para o seu webhookUrl com o conteúdo completo da mensagem — remetente, assunto, corpo e anexos.

Passo 4: Usar o MCP Skill (Agentes Claude)

Se o seu agente corre em Claude, instale o MCP skill do AgentSend para que o Claude tenha ferramentas nativas para enviar emails, verificar caixas de entrada e ler threads — sem código webhook adicional.

Consulte a documentação do MCP skill para todas as ferramentas disponíveis.

Perguntas Frequentes

Um agente de IA pode ter o seu próprio endereço de email?

Sim. O AgentSend permite provisionar uma caixa de email real para qualquer agente de IA de forma programática — sem registo humano. O seu agente obtém um endereço real @agentsend.io (ou o seu domínio personalizado).

Com que frameworks de agentes é compatível o AgentSend?

O AgentSend funciona com qualquer framework que possa fazer pedidos HTTP ou usar ferramentas MCP — incluindo Claude, LangChain, CrewAI, AutoGen e OpenAI Assistants.

O AgentSend é gratuito?

O AgentSend tem um plano gratuito: até 10 emails por dia por caixa de entrada. Os planos pagos começam em $9/mês para 1.000 emails/dia com domínios personalizados.

Pronto para dar uma caixa de email ao seu agente?

Plano gratuito disponível. Sem cartão de crédito. O seu endereço de email fica ativo em menos de 30 segundos.

Comment Donner une Boîte Mail à votre Agent IA

Équipe AgentSend · 20 mars 2026 · 8 min de lecture
Lire en : EN ES FR DE JA ZH PT

Les agents IA peuvent écrire du code, naviguer sur le web et exécuter des workflows complexes — mais la plupart ne peuvent toujours pas recevoir un simple email. AgentSend résout ce problème. C'est une infrastructure email conçue spécifiquement pour les agents : provisionnez une vraie boîte mail programmatiquement, recevez des messages via webhook et répondez via l'API REST.

Étape 1 : Créer un Compte et Obtenir une Clé API

Rendez-vous sur agentsend.io/auth/signup et créez un compte gratuit. Après l'inscription, générez une clé API dans Paramètres → Clés API.

Étape 2 : Créer une Boîte Mail

Une boîte mail est le conteneur de l'adresse email de votre agent. Créez-en une avec un seul appel API :

POST /inboxes
curl -X POST https://agentsend.io/inboxes \
  -H "Authorization: Bearer VOTRE_CLE_API" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "mon-agent-recherche",
    "webhookUrl": "https://votre-serveur.com/webhooks/email"
  }'

Votre agent a maintenant une vraie adresse email : mon-agent-recherche@agentsend.io

Étape 3 : Gérer les Emails Entrants via Webhook

Quand quelqu'un envoie un email à votre agent, AgentSend effectue un HTTP POST vers votre webhookUrl avec le payload complet du message — expéditeur, sujet, corps et pièces jointes.

Étape 4 : Utiliser le MCP Skill (Agents Claude)

Si votre agent tourne sur Claude, installez le MCP skill AgentSend pour que Claude dispose d'outils natifs pour envoyer des emails, consulter les boîtes mail et lire les fils de discussion — sans code webhook supplémentaire.

Consultez la documentation du MCP skill pour tous les outils disponibles.

Questions Fréquentes

Un agent IA peut-il avoir sa propre adresse email ?

Oui. AgentSend vous permet de provisionner une vraie boîte mail pour n'importe quel agent IA de façon programmatique — sans inscription humaine. Votre agent obtient une vraie adresse @agentsend.io (ou votre domaine personnalisé).

Avec quels frameworks d'agents AgentSend est-il compatible ?

AgentSend fonctionne avec n'importe quel framework pouvant effectuer des requêtes HTTP ou utiliser des outils MCP — dont Claude, LangChain, CrewAI, AutoGen et OpenAI Assistants.

AgentSend est-il gratuit ?

AgentSend dispose d'un plan gratuit : jusqu'à 10 emails par jour par boîte mail. Les plans payants démarrent à 9 $/mois pour 1 000 emails/jour avec des domaines personnalisés.

Prêt à donner une boîte mail à votre agent ?

Plan gratuit disponible. Sans carte de crédit. Votre adresse email est active en moins de 30 secondes.

如何为 AI 智能体配置专属电子邮箱

AgentSend 团队· 2026年3月20日· 约 8 分钟阅读
其他语言阅读： EN ES FR DE JA ZH PT

AI 智能体可以编写代码、浏览网页并执行复杂的工作流 —— 但大多数仍然无法接收一封简单的电子邮件。AgentSend 解决了这个问题。这是专为智能体设计的电子邮件基础设施：以编程方式配置真实邮箱、通过 Webhook 接收消息、并使用 REST API 进行回复。

第一步：创建账户并获取 API Key

访问 agentsend.io/auth/signup 创建免费账户。免费套餐每天最多支持 10 封邮件。注册后，在设置 → API Keys 中生成一个 API Key。

第二步：创建邮箱（Inbox）

Inbox 是智能体电子邮件地址的容器。通过一次 API 调用即可创建：

POST /inboxes
curl -X POST https://agentsend.io/inboxes \
  -H "Authorization: Bearer 你的API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "my-research-agent",
    "webhookUrl": "https://your-server.com/webhooks/email"
  }'

你的智能体现在拥有了一个真实的电子邮件地址：my-research-agent@agentsend.io

第三步：通过 Webhook 处理接收邮件

当有人给智能体发送邮件时，AgentSend 会向你的 webhookUrl 发送 HTTP POST 请求，包含完整的消息内容 —— 发件人、主题、正文和附件。

第四步：使用 MCP Skill（Claude 智能体）

如果你的智能体运行在 Claude 上，安装 AgentSend MCP Skill 是最快的方式。Claude 将获得原生工具来发送邮件、查看邮箱和阅读邮件线程 —— 无需额外的 Webhook 代码。

所有可用工具请参阅 MCP Skill 文档。

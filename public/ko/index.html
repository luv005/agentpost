2

인박스 생성

API 한 번의 호출로 고유 주소가 포함된 에이전트 전용 이메일 인박스를 생성합니다.

3

송신 & 수신

에이전트가 이메일을 보내고, 웹훅으로 답장을 받고, 전체 대화 스레드를 추적합니다.

에이전트에게 필요한 모든 것

이메일에 AI를 쓰는 게 아닙니다. AI에 이메일을 쓰는 겁니다.

📨

송신 & 수신

완전한 양방향 이메일. 발신은 AWS SES, 수신은 내장 SMTP 서버. 큐 기반 비동기 처리.

💬

자동 스레딩

RFC 5322 준수 대화 스레딩. Message-ID, In-Reply-To, References를 통한 메시지 자동 그룹화.

⚡

실시간 웹훅

이메일 수신 시 즉시 알림. HMAC-SHA256 서명 페이로드와 자동 재시도 및 백오프.

🌐

커스텀 도메인

자동 DKIM 및 SPF 인증으로 자체 도메인을 사용하세요. 에이전트가 your-brand.com에서 발송합니다.

📎

첨부 파일

최대 10MB 파일 업로드 및 첨부. S3 기반 스토리지와 사전 서명된 다운로드 URL.

🛡️

반송 추적

반송, 불만, 배달 확인에 대한 SES 이벤트 자동 처리.

🔑

API 키 인증

간편한 Bearer 토큰 인증. SHA-256 해시 저장, 계정 및 인박스별 속도 제한.

🧩

MCP Skill

Claude, Cursor 및 MCP 호환 AI 에이전트를 위한 드롭인 MCP 서버. npx @agentsend/mcp.

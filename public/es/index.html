📨

Enviar y recibir

Email bidireccional completo. AWS SES para envio, servidor SMTP integrado para recepcion. Procesamiento asincrono basado en colas.

💬

Hilos automaticos

Agrupacion de conversaciones compatible con RFC 5322. Mensajes agrupados automaticamente via Message-ID, In-Reply-To y References.

⚡

Webhooks en tiempo real

Notificaciones instantaneas cuando llegan emails. Payloads firmados con HMAC-SHA256 con reintentos automaticos y backoff.

🌐

Dominios personalizados

Usa tu propio dominio con verificacion automatica de DKIM y SPF. Tu agente envia desde tu-marca.com.

📎

Archivos adjuntos

Sube y adjunta archivos de hasta 10MB. Almacenamiento en S3 con URLs de descarga prefirmadas.

🛡️

Seguimiento de rebotes

Procesamiento automatico de eventos SES para rebotes, quejas y confirmaciones de entrega.

🔑

Autenticacion por clave API

Autenticacion simple con bearer token. Almacenamiento con hash SHA-256 y limitacion de tasa por cuenta y por bandeja.

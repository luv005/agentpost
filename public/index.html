Send, receive, and manage email conversations programmatically. Full two-way email with threading, webhooks, and custom domains.

View API Docs GitHub 
import { AgentSend } from "@agentsend/sdk";

const client = new AgentSend({ apiKey: "ask_..." });

// Create an inbox
const inbox = await client.inboxes.create({
  displayName: "Support Bot"
});

// Send an email
await client.messages.send(inbox.id, {
  to: ["user@example.com"],
  subject: "Hello from my AI agent",
  bodyText: "This email was sent by an AI agent."
});

// List conversation threads
const threads = await client.threads.list(inbox.id);

Everything your agent needs

✉

Send & Receive

Full two-way email via AWS SES (outbound) and built-in SMTP server (inbound). Queue-based async processing.

💬

Threading

RFC 5322 compliant conversation threading. Auto-resolves threads via Message-ID, In-Reply-To, and References headers.

🔌

Webhooks

Real-time notifications when emails arrive. HMAC-SHA256 signed payloads with automatic retry and exponential backoff.

🌐

Custom Domains

Add your own domain with DKIM/SPF verification via SES. Create inboxes on any verified domain.

📎

Attachments

Upload files to S3 and attach to outbound emails. Automatic MIME encoding with presigned download URLs.

🔐

API Key Auth

Simple API key authentication. SHA-256 hashed storage, rate limiting per account and per inbox.

智能体所需的一切

不是用 AI 管理你的邮件，而是为你的 AI 提供邮件。

📨

收发邮件

完整的双向邮件功能。AWS SES 负责发送，内置 SMTP 服务器负责接收。基于队列的异步处理。

💬

自动会话归组

符合 RFC 5322 标准的会话归组。通过 Message-ID、In-Reply-To 和 References 自动关联消息。

⚡

实时 Webhook 回调

邮件到达时即时通知。HMAC-SHA256 签名的负载，自动重试与退避机制。

🌐

自定义域名

使用你自己的域名，自动完成 DKIM 和 SPF 验证。让智能体从 your-brand.com 发送邮件。

📎

附件支持

上传和附加最大 10MB 的文件。基于 S3 的存储，提供预签名下载链接。

🛡️

退信追踪

自动处理 SES 事件，包括退信、投诉和送达确认。

🔑

API 密钥认证

简洁的 Bearer Token 认证。SHA-256 哈希存储，支持按账户和按邮箱的速率限制。
